TESTING INSTRUCTIONS
================================================================================

To run the application:
  1. cd java-implementation
  2. mvn javafx:run
     (Note: Requires Maven and JavaFX. If Maven not installed,
      the build.sh script can compile without running)

To run unit tests:
  1. cd java-implementation
  2. mvn test
     (Note: Requires Maven and JUnit dependencies)

Manual Testing:
  1. Start application
  2. Browse venues on home page
  3. Click on a venue to view details
  4. Select time slots (they turn green)
  5. Click checkout button
  6. Review order and select payment method
  7. Click confirm payment
